A promise-based frontend for [[https://github.com/magit/magit][magit-section]].

* Requirements

- Emacs 28.1 or later
- magit-section 2.90 or later

* Installation

** Using leaf

#+begin_src emacs-lisp
(leaf promise-magit-section :ensure t)
#+end_src

** Using use-package

#+begin_src emacs-lisp
(use-package promise-magit-section
  :ensure t)
#+end_src

** Manual Installation

Clone this repository and add it to your =load-path=:

#+begin_src emacs-lisp
(add-to-list 'load-path "/path/to/promise-magit-section.el")
(require 'promise-magit-section)
#+end_src

** Development Setup

1. Install [[https://github.com/cask/cask][Cask]] for dependency management:

   #+begin_src shell
   # Using Homebrew
   brew install cask

   # Or manual installation
   git clone https://github.com/cask/cask ~/.cask
   export PATH="$HOME/.cask/bin:$PATH"
   #+end_src

2. Run tests:

   #+begin_src shell
   make test
   #+end_src
